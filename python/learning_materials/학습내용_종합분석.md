### 2단원: NumPy - 수치 계산의 기초

#### 2-1. NumPy ndarray 기초

**핵심 개념:**
- `ndarray`: NumPy의 핵심 자료구조 (N-dimensional array)
- 벡터 및 행렬 연산의 효율적인 처리

**주요 함수:**
```python
np.array()      # 배열 생성
np.zeros()      # 0으로 채워진 배열
np.ones()       # 1로 채워진 배열
np.arange()     # 연속된 숫자 배열
```

**데이터 타입:**
- `dtype`: int32, int64, float64, bool 등
- `astype()`: 타입 변환

**배열 연산:**
- 같은 크기 배열 간 연산 (element-wise)
- 브로드캐스팅: 스칼라와 배열 간 연산

---

#### 2-2. Array 인덱싱

**1차원 인덱싱:**
```python
arr[5]          # 단일 요소
arr[5:8]        # 슬라이싱
arr[5:8] = 12   # 값 할당
```

**2차원 인덱싱:**
```python
arr2[2, :]      # 특정 행 전체
arr2[:, 3]      # 특정 열 전체
arr2[1:3, 2:4]  # 범위 슬라이싱
```

**불리언 인덱싱 (Boolean Indexing):**
```python
mask = arr > 50
arr[mask]                    # 조건 만족하는 요소만
arr[(arr > 10) & (arr < 20)] # 복합 조건
```

**핵심 포인트:**
- 불리언 마스크를 사용한 필터링이 매우 강력함
- `&` (and), `|` (or) 연산자로 복합 조건 생성

---

#### 2-3. Array 관련 함수

**수학 함수 (각 성분에 적용):**
```python
np.abs()        # 절댓값
np.sqrt()       # 제곱근
np.square()     # 제곱
np.log(), np.log10()  # 로그
np.ceil(), np.floor() # 올림, 내림
```

**두 배열 간 함수:**
```python
np.maximum()    # 위치별 최댓값 선택
np.minimum()    # 위치별 최솟값 선택
np.add(), np.subtract()  # 덧셈, 뺄셈
```

**통계 함수:**
```python
arr.sum()       # 합계
arr.mean()      # 평균
arr.std()       # 표준편차
arr.var()       # 분산
arr.min(), arr.max()  # 최솟값, 최댓값
arr.argmin(), arr.argmax()  # 최솟값/최댓값 인덱스
arr.cumsum()    # 누적합
```

**정렬 및 기타:**
```python
np.sort()       # 정렬
np.unique()     # 중복 제거
```

**axis 개념 (매우 중요!):**
- `axis=0`: 행 방향 (↓) - 각 열에 대해 계산
- `axis=1`: 열 방향 (→) - 각 행에 대해 계산

```python
arr.sum(axis=0)  # 각 열의 합
arr.sum(axis=1)  # 각 행의 합
```

### 3단원: Pandas - 데이터 분석의 핵심

#### 3-1. Pandas 자료구조

**Series:**
- 1차원 라벨링된 배열
- 인덱스와 값으로 구성
```python
pd.Series([4,7,-5,3])
pd.Series(dict)  # 딕셔너리로 생성
```

**DataFrame:**
- 2차원 테이블 (행과 열)
- 여러 Series의 모음
```python
pd.DataFrame(data)
df.index        # 행 인덱스
df.columns      # 열 이름
df.values       # 값 (numpy array)
```

**주요 메서드:**
```python
df.describe()   # 기본 통계량
df.head()       # 처음 5개
df.tail()       # 마지막 5개
df.shape        # 차원
```

---

#### 3-2. DataFrame 인덱싱

**컬럼 선택:**
```python
df['year']          # 단일 컬럼 (Series)
df[['year', 'name']] # 여러 컬럼 (DataFrame)
```

**컬럼 추가/삭제:**
```python
df['new_col'] = values
del df['col_name']
```

**loc (라벨 기반):**
```python
df.loc['row_label']           # 특정 행
df.loc['r1':'r3', 'col1']     # 범위 선택
df.loc[:, ['c1', 'c2']]       # 컬럼 선택
```

**iloc (위치 기반):**
```python
df.iloc[0]          # 첫 번째 행
df.iloc[0:3, 1:3]   # 슬라이싱
df.iloc[[0,2], [1,3]]  # 특정 위치들
```

**불리언 인덱싱:**
```python
df[df['age'] > 30]
df[(df['age'] > 20) & (df['score'] >= 80)]
df.loc[df['city'] == 'Seoul', ['name', 'age']]
```

**핵심 차이:**
- `loc`: 라벨/이름 사용 (포함적 - 끝 포함)
- `iloc`: 정수 위치 사용 (배타적 - 끝 미포함)

---

#### 3-3. DataFrame 조작

**날짜 인덱스:**
```python
pd.date_range('20170701', periods=6)
```

**결측치 처리 (매우 중요!):**
```python
df.isnull()         # NaN 확인
df.dropna(how='any') # NaN 있는 행 삭제
df.dropna(how='all') # 전체가 NaN인 행만 삭제
df.fillna(value)    # NaN을 특정 값으로 채우기
```

**행/열 삭제:**
```python
df.drop(index)          # 행 삭제
df.drop('col', axis=1)  # 열 삭제
df.drop(['c1','c2'], axis=1)  # 여러 열 삭제
```

---

#### 3-4. DataFrame 데이터 분석 함수

**집계 함수:**
```python
df.sum()        # 합계
df.mean()       # 평균
df.sum(axis=0)  # 각 열의 합
df.sum(axis=1)  # 각 행의 합
df.mean(axis=1, skipna=False)  # NaN 무시 안 함
```

**상관계수:**
```python
df['A'].corr(df['B'])  # 두 컬럼 간 상관계수
df.corr()              # 전체 상관 행렬
```

**정렬:**
```python
df.sort_index(axis=0)   # 인덱스 기준 정렬
df.sort_index(axis=1)   # 컬럼 기준 정렬
df.sort_values(by='col')  # 값 기준 정렬
df.sort_values(by=['c1','c2'], ascending=False)  # 내림차순
```

**데이터 탐색:**
```python
df['col'].unique()      # 고유값
df['col'].value_counts()  # 값 빈도수
df['col'].isin(['A', 'B'])  # 값 포함 여부
```

**apply + lambda:**
```python
df['col'].apply(lambda x: x * 2)
df.apply(lambda row: row['A'] + row['B'], axis=1)
```

**핵심 포인트:**
- `apply()`는 사용자 정의 함수를 각 행/열에 적용
- `axis=1`로 행 전체에 접근 가능
- lambda로 간단한 변환 작성
