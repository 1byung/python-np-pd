# 데이터 분석 학습 자료

GitHub 저장소 [ilovejwoo1004-bit/ai](https://github.com/ilovejwoo1004-bit/ai)의 학습 자료를 Python 파일 형식으로 정리한 폴더입니다.

## 📁 폴더 구조

```
learning_materials/
├── notebooks/          # Jupyter Notebook을 Python 스크립트로 변환한 파일들
├── movielens/         # MovieLens 데이터셋 (필요 시 별도 다운로드)
├── docs/              # 설치 가이드 및 문서
└── README.md          # 이 파일
```

## 📚 학습 자료 목록

### 1단원: 데이터 분석 기초
- `1-01_데이터분석이란.py` - 데이터 분석의 개념과 빅데이터 탐색적 분석
- `1-02_파일_데이터_로드.py` - Numpy와 Pandas를 사용한 파일 읽기

### 2단원: NumPy
- `2-01_numpy_ndarray.py` - NumPy 배열(ndarray) 기초
- `2-02_array_인덱싱.py` - 배열 인덱싱과 슬라이싱
- `2-03_array_관련_함수_사용.py` - NumPy 함수 활용
- `2-04_numpy를_사용한_데이터_분석_MovieLens.py` - MovieLens 데이터셋 분석 실습

### 3단원: Pandas
- `3-01_Pandas_자료구조_Series와_DataFrame.py` - Series와 DataFrame 기본
- `3-02_DataFrame_인덱싱.py` - DataFrame 인덱싱 (.loc, .iloc)
- `3-03_DataFrame_조작하기.py` - DataFrame 조작 및 NaN 처리
- `3-04_DataFrame_데이터_분석용_함수_사용.py` - 통계 함수, 정렬, apply
- `3-05_pandas_문제.py` - Lending Club Loan 데이터셋 분석 문제
