# 포트폴리오 가이드

**목표**: 취업용 포트폴리오 3~5개 완성
**메인 타겟**: 데이터 분석가 → ML 엔지니어 → 데이터 사이언티스트

---

## 🎯 포트폴리오 전략

### 핵심 원칙
1. **실용성**: "실제로 쓸모 있는가?"
2. **차별화**: "남들이 안 한 것인가?"
3. **완성도**: "제대로 마무리했는가?"
4. **스토리**: "왜 이걸 했는지 설명 가능한가?"

### 개수 전략
```
❌ 10개 미완성 프로젝트
✅ 3개 완벽한 프로젝트

채용 담당자는 3개만 봄!
```

# 🎧 공항 소음 저감 시스템

- 소음 패턴 분석 및 예측
- Active Noise Cancellation으로 평균 **15dB 감소**
- 실시간 처리 가능한 경량 모델

## 🛠️ 기술 스택
- Python, Librosa, NumPy, SciPy
- PyTorch (LSTM)
- Streamlit

## 📊 주요 결과
- 평균 소음 감소: **15.3dB**
- 특정 주파수 대역: **최대 25dB 감소**
- 처리 속도: **실시간 (< 100ms)**
